# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-03-14

### Added
- `SortedList`: a list that maintains sorted order on insert
- `SortedKeyList`: sorted list with a key function for custom ordering
- `SortedDict`: dict with keys maintained in sorted order
- `SortedSet`: set with elements maintained in sorted order
- Full Python 3.10+ support with zero dependencies
- Type annotations throughout (PEP 561, `py.typed` marker)
- Note: package renamed from `collate` (PyPI name was taken) to `sortsmith`
